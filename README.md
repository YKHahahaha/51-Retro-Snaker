# 51-Retro-Snaker
基于金沙滩51单片机的贪吃蛇小游戏

游戏说明：上下左右键控制方向；

          按回车键暂停或继续；
          
          吃到一个食物记一分，显示在右上角计分板；
          
          撞到自身，游戏失败
          
          吃完26个球，游戏方算成功

开发环境：KST-51 v1.3.2  keil5

下载全部文件，打开“贪吃蛇.uvproj”即可
